# Snake Battle Royale — Dindon Design

## Quick Start
1. Ouvrir `src/index.html` dans un browser
2. Appuyer sur **2**, **3** ou **4** pour choisir le nombre de joueurs
3. Jouer !

## Contrôles

| Joueur | Touches |
|--------|---------|
| J1 | Z Q S V |
| J2 | Flèches directionnelles |
| J3 | I J K L |
| J4 | Numpad (8, 4, 5, 6) |

## Règles
- Manger la nourriture = +1 point, +1 segment
- Collision mur = mort
- Collision corps = mort
- Collision tête-à-tête = les deux meurent
- Dernier survivor gagne !
