# Game Design - Snake Battle Royale

## Concept
Un Snake compétitif en mode "Battle Royale" : 2 à 4 joueurs s'affrontent sur un même plateau. Le dernier serpent vivant remporte la partie.

## Gameplay
- **Contrôles** : Clavier dédié par joueur
- **Règles** : Manger pastilles = longueur + score. Heurter mur/corps/autrui = élimination.
- **Victoire** : Dernier survivant

## Multiplayer
- J1: Z Q S V (Haut, Gauche, Bas, Droite)
- J2: Flèches directionnelles
- J3: I J K L
- J4: Numpad (8,4,5,6)

## Power-ups
- 🟢 Nourriture: +3 segments, +1 point
- 🔴 Bombe: Élimine le joueur (piège)
- ⚡ Boost: Vitesse x2 pendant 5s

## Graphics
Canvas 2D, style néon/rétro, particules, 60 FPS
