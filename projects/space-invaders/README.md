# Space Invaders — Dindon Design v0.1

## Quick Start
1. Ouvrir `src/index.html` dans un browser
2. Appuyer sur SPACE pour démarrer
3. ← → pour bouger, SPACE pour tirer

## Gameplay
- Détruisez les aliens pour marquer des points
- Évitez leurs tirs
- Progression par niveaux
- 3 vies

## Controls
- **← →** Déplacement
- **SPACE** Tirer / Démarrer

## Tech
- Vanilla JS, Canvas 2D
- Pas de dépendances
- Sprites dessinés via code
